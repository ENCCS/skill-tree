# USE1.1-B Command Line Interface
# Background
# Aim

  * To use the interactive shell efficiently in order to navigate through directories, find files and retrieve information.
  * To perform file operations as (re-)naming files, creating directories.

# Outcomes
  *  execute frequently used commands, e.g. to navigate the file system, copy, rename, and delete files, view the contents of files, and to get detailed help for the usage of a command with all its options 
  *  use wildcards to select or filter e.g. files 
  *  login remotely to cluster nodes using e.g. SSH with password or SSH key authentication 
  *  access local and remote files in remote sessions 
  *  check disk quotas commonly used to limit the amount of disk space available for the user
