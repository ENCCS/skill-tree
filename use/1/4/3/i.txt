# USE1.4.3-I Spack

# Background

The intermediate level of Spack deals with the setup of software using Spack.
This activity can be performed by users and empowers them to manage the appropriate software on supercomputers.

# Aim
  * To manage a software repository using Spack.

# Outcomes

  * Installation of Spack
  * Build software using Spack
    - Setup of compilers
    - Show installed compilers
    - Configuration of variants
  * Manage the installation directories
  * Describe relevant directories for the Spack installation
